# The Daily Runners Android Application

The Daily Runners is the application of the runners.
Here you can see chronological approach the races and all the information you want to know, location, date, value, geographic location, kit, registration and more.
A filter racing for you is easier to search for marathons, half marathons, trail and other categories. Available only El Salvador races for now.

## Google Play 
https://play.google.com/store/apps/details?id=com.turtlesoftware.thedailyrunners.main
